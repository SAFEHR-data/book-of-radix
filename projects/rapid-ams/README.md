# RAPID-AMS

We will build three sequential research protocols within the same patient population for the same ultimate goal of better antibiotic prescribing.&#x20;

Each stage seeks to demonstrate better prescribing by providing information at the time of prescribing on

1. Local antibiotic guidelines
2. Historical anti-microbial resistance
3. Predicted anti-microbial resistance&#x20;
