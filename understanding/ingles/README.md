# Inglés

> **I**n‑situ **N**udges for **G**uidelines, **L**ocal resistance, **E**stimated **S**usceptibility

We will build three sequential research protocols within the same patient population for the same ultimate goal of better antibiotic prescribing.

Each stage seeks to demonstrate better prescribing by providing information at the time of prescribing on

1. Inglés-**G**uidelines
2. Inglés-**R**esistance
3. Inglés-**E**stimated Susceptibility
