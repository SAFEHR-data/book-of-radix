# The Book of RADIX

**RADIX** is the **R**eal-time **A**ntimicrobial **D**igital **I**ntervention e**X**change programme run jointly by UCLH, UCL and HSL. The documentation is split between specific projects and general resources.

### Projects

We have split our work into 9 projects across 3 categories: [foundations](./#foundations), [decisions](./#decisions) and [understanding](./#understanding). Our project names are inspired by the [Camino de Santiago](https://en.wikipedia.org/wiki/Camino_de_Santiago) and its variants as we see the task of using data to improve health as worthy journey!
